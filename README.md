## Scrap Puppet

### First Step
Certify that RabbitMq and Postgres are up and running. Use this repo to build the images (scrap folder):
[Containers](https://github.com/guimassoqueto/containers)


1. init husky
```shell
make h
```

2. create .env
```shell
make env
```

3. install dependencies
```shell
make i
```

4. init app
```shell
make a
```
